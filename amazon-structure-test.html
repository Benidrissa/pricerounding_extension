ðŸ“‹ How The New Generic Processing Works

1. Primary Approach: Offscreen-First

  • Look for .a-offscreen element (contains full price text like "$49.99")
  • Process this text using regular price processing patterns
  • Update the offscreen element with rounded text
  • Update visual elements (symbol, whole, fraction) to match

2. Fallback: Visual Elements

  • If no offscreen element, use .a-price-whole and .a-price-fraction
  • Combine them to get the original price
  • Round and update both visual and offscreen elements

3. Format Preservation

  • Preserves original formatting (e.g., "49." vs "49")
  • Updates fraction to "00" for rounded prices
  • Maintains accessibility with proper offscreen text
